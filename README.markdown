# Getting started
After cloning the repo, run `rake` to initialise and update the submodule bundles (or manually run `git submodule init; git submodule update`).

Then link:

* `repo` to `~/.vim`
* `repo/vimrc` to `~/.vimrc`
* `repo/gvimrc` to `~/.gvimrc`
